111年第一學期科學計算導論的期末專案，我們的主題為「伯特蘭悖論」\
伯特蘭悖論的問題為：**考慮一個內接於圓的等邊三角形。若隨機選圓上的弦，則此弦的長度比三角形的邊較長的機率為何？**

我們將透過 Matlab 實作三個不同的方法，分別為：隨機端點法、隨機半徑法、隨機中點法
